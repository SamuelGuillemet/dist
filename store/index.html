Pack Agri-Mini

Ce pack a pour but de vous faire découvrir notre solution. Il contient les capteurs de base pour commencer à mesurer les données de votre exploitation. Il est a destination des très petits exploitants.

Pack Agri-Standard

Ce pack vous permet de mesurer les données de base de votre exploitation. Il est a destination des petits exploitants.

Pack Agri-Performant

Ce pack est le plus populaire. Il vous permet de mesurer toutes les données de vos exploitations. Il est a destination des exploitants moyens.

Pack Agri-Excellence

Ce pack est le plus complet. Il vous permet de mesurer toutes les données de vos exploitations. Il est a destination des grands exploitants et des coopératives.
